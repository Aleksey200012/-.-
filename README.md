                                                                                           Описание программы для сериализации двусвязного списка
1) Включение заголовков и структура узла.
Подключаются стандартные заголовочные файлы: iostream, fstream, vector, string, memory.
Определена структура ListNode:
 1. data: строка с данными узла.
 2. rand: указатель на другой ListNode (возможен nullptr).
 3. конструктор ListNode(const std::string& d) инициализирует data и устанавливает rand в nullptr.

2) Точка входа и константы файлов.
В main объявляются две константы имен файлов:
  1. inputFileName = "Inlet.in"
  2. outputFileName = "outlet.out"
     
3) Открытие входного файла через std::ifstream. Если открыть файл не удалось, выводится сообщение об ошибке: "Не удалось открыть файл Inlet.in" и программа завершается с кодом 1.
   
4) Чтение данных узлов. Для этого объявим два вектора:
  1. nodeData: хранит строковые данные каждого узла.
  2. randIntices: хранит целочисленные индексы для поля rand каждого узла (или -1 для отсутствия).
     
Далее, считываем построчно файл Inlet.in, где:
  1. Каждая строка ожидается в формате "данные;индекс".
  2. По строке ищется символ ';'. Если разделитель не найден — строка пропускается.
  3. Часть до ';' сохраняется как dataPart, часть после ';' как indexPart.
  4. Пробуем преобразовать indexPart в int с помощью std::stoi. При любой ошибке устанавливается randIdx = -1.
  5. Добавляются значения в соответствующие векторы.

5) Проверяем количества узлов, где:
  1. n = nodeData.size().
  2. Если число узлов превышает 1 000 000, выводится сообщение "Превышено допустимое количество узлов." и программа завершается кодом 1.
     
6) Создание и заполнение узлов.
Создаем вектор узлов, в котором nodes имеет размер n и инициализируется nullptr. После этого, создаем сами узлы по принципу: Для каждого i создается новый ListNode с данными nodeData[i], и сохраняется в nodes[i].

7) Связываем узлы друг с другом через поле rand.

8) Сериализуем двусвязный список в бинарный файл outlet.out. Для этого откроем выходной бинарный файл с флагом std::ios::binary. Если открыть файл не удалось, то выводится сообщение об ошибке, освобождаются все узлы через delete, и программа завершается кодом 1.
Далее записываем данные по следующему алгоритму:
  1. Сначала записывается общее количество узлов as uint64_t nodeCount = n.
  2. Затем для каждого узла:
    - Длина данных dataLen = nodeData[i].size() записывается как uint64_t.
    - Сама строка данных записывается как последовательность байтов через fout.write(nodeData[i].c_str(), dataLen).
    - Индекс rand для узла записывается как int randIdxWrite = randIntices[i].
После записи все узлы удаляются через delete, файл закрывается и происходит полное завершение программы.
